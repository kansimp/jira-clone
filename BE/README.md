# Run locally

## Ensure

```sh
node >= v22.14.0
pnpm >= v10.7.0
```

## CMD to dev

```sh
pnpm start:dev
```

# Run container

## Ensure

```sh
docker >= v27.4.0
```

## CMD to run

```sh
pnpm start:docker
```

## CMD to stop

```sh
pnpm stop:docker
```
